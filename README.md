## 🎯 Repository Purpose

The following labs show you how to work with Bob and explore key capabilities. Your facilitors are there for you to guide you through the process and help with the setup.

## 🤖 List of Labs

We recommend you to create a branch before starting using Bob with this repository, so in case of an error, or to repeat the lab, you can always get back to the `main` branch.

### 🛰️ Java Modernization

* [Modernize a Spring Boot 1.5 / Java 8 Banking API](java-modernization): A lab to show you how to modernize Java with Bob using the _IBM Bob Premium Package for Java Modernization_.  

### 🖥️ COBOL Modernization

* [Discover and modernize a COBOL banking application (CBSA)](cobol-modernization): Inventory, document, and evolve a COBOL application using the _IBM Bob Premium Package for Z_.

### 🛠️ Python and React Development Tools

* [Prototype a core banking system](python-react-development): Explore a Python/FastAPI backend and build a modern React UI fast with a Carbon-style MCP.

## 💡 Tips for Working with Bob

- **Choose the right mode:** Agent, Plan, Ask, or premium development modes - each has its strengths
- **Be specific:** Clear, detailed prompts yield better results
- **Iterate:** Don't expect perfection on the first try - refine as needed
- **Document everything:** Capture your process for others to learn from
- **Share challenges:** Failed attempts are valuable learning opportunities
