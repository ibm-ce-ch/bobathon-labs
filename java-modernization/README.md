# Modernize a Core Banking Application

Welcome!

Use IBM Bob to modernize a prototype core banking system end to end: understand a legacy Java/Spring backend and its banking domain through documentation and architecture analysis, upgrade it from Java 8 / Spring Boot 1.5 to Java 21 / Spring Boot 3.x, then build a modern full-stack banking dashboard on top of the modernized API.

A hands-on, guided lab in two parts.

| Part | What you do |
|---|---|
| **[1 · Modernize the Backend](Bob_V2_lab_part1.md)** | Chat with Bob to assess the legacy Spring Boot 1.5 banking application, generate modernization specs, and migrate it to Java 21, Spring Boot 3.x, Jakarta, and PostgreSQL. |
| **[2 · Build the Dashboard](Bob_V2_lab_part2.md)** | Use Bob to generate a modern Vaadin 24 banking dashboard in Java that consumes the modernized REST API using typed, contract-driven integration. |

> Start with [Part 1](Bob_V2_lab_part1.md).
